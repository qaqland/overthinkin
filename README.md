# overthinkin

突然有天意识到，我写的低成本小工具们，可能永远不会得到下一次修改或编译的机会。
因此不如把大家集合起来抱团维护，成为小小的 Monorepo 组合起来。

小工具们有以下特点：

- 成本很低，从需求构思到完成代码 MVP 基本不会超过 2 周；
- 代码简单，基本都是大学生 C 语言和水群能学到的知识；
- 重在参与，它们被创造出来的首要目的是学习和快乐，没有什么使命。

项目名受到《第五人格》中一位游戏好友的 ID 启发。希望大家天天开心。

## 协议

纯自研默认 MIT，或者受到依赖库的限制自动变为 GPL 不等。

## AIGC

项目不接收没有感情的 AI 建议，只为人类或其它有感情的虚拟服务。
